📋 Panduan Lengkap Instalasi

📥
Download & Install APK
Tap tombol "Download E-CTBC" di atas
Tunggu proses download selesai (sekitar 15MB)
Buka file APK dari notifikasi atau folder Downloads
Ikuti wizard instalasi yang muncul di layar
🔒
Izinkan Sumber Tidak Dikenal
Buka Pengaturan → Keamanan & Privasi
Aktifkan "Sumber Tidak Dikenal" atau "Unknown Sources"
Untuk Android 8+: Pilih "Allow from this source" saat install
Pastikan centang "Install apps from external sources"
🔔
Aktifkan Notifikasi Push
Saat pertama buka aplikasi, izinkan semua notifikasi
Tap "Allow" atau "Izinkan" pada popup
Manual: Pengaturan → Apps → E-CTBC → Notifications → ON
Aktifkan juga "Show on lock screen" untuk notifikasi penting
⚙️
Izinkan Background Activity
Buka Pengaturan → Battery atau Battery Optimization
Cari aplikasi E-CTBC → Pilih "Don't optimize"
Samsung: Apps → E-CTBC → Battery → "Allow background activity"
Xiaomi/MIUI: Apps → E-CTBC → "Autostart: ON" & "Background app refresh: ON"
⚠️ Pengaturan Penting untuk Performa Optimal
Aplikasi E-CTBC memerlukan izin notifikasi dan background activity untuk memberikan layanan real-time yang optimal. Tanpa kedua pengaturan ini, beberapa fitur mungkin tidak berfungsi dengan sempurna. Pastikan mengikuti semua langkah di atas!
